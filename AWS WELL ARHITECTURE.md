**AWS SECURITY HUB** 
AWS Security Hub is a **security dashboard** for your AWS environment.  
It collects security findings from different AWS services (like GuardDuty, IAM Access Analyzer, Inspector, etc.) and puts them all in **one place** so you can easily see and fix security issues.
 NOTE:
 **AWS security hub uses AWS config to perform** security checks. aws config must be enabled on all accounts for this to take effect.**
